A full-stack MERN (MongoDB, Express, React, Node.js) Job Portal application where recruiters can post jobs and candidates can apply for them.

---

## Features

- User Authentication (JWT)
- Role-Based Authorization (Admin / User)
- Job Posting & Browsing
- Job Application System
- Admin Dashboard
- Profile Update with Image Upload

---

## Tech Stack

Frontend:
- React (Vite)
- Redux Toolkit
- Tailwind CSS
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Multer
- Cloudinary

---

## How To Run Locally

Backend:

cd backend  
npm install  
npm run dev  

Frontend:

cd frontend  
npm install  
npm run dev
